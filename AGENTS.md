# AGENTS.md

Правила для всех задач в этом репозитории:

1. В конце каждой завершённой задачи обязательно выводить:
   - Commit summary (EN)
   - Commit description (EN)
   - PR title (RU)
   - PR description (RU) в формате Мотивация / Описание / Тестирование
   - список изменённых файлов
   - список того, что не было проверено запуском

2. Не писать, что тесты запускались, если они реально не запускались.

3. Если задача многоэтапная, в PR description явно указывать:
   - что сделано на этом этапе
   - что отложено на следующий этап

4. Все описания для пользователя и PR готовить на русском, кроме краткого commit summary/description, если пользователь явно не попросил иначе.
